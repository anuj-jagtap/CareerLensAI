# Features

## Module 1: Job Market Analytics

* Most demanded IT roles
* Most demanded skills
* Skill trends over time
* Location-wise job demand
* Salary insights

## Module 2: Resume Intelligence

Input:

* Resume PDF

Output:

* Extracted skills
* Education details
* Projects
* Certifications

## Module 3: Job Role Recommendation

Suggest suitable roles such as:

* Data Analyst
* Business Analyst
* Data Scientist
* BI Analyst
* SAS Programmer

## Module 4: Job Match Score

Calculate alignment between:

* Candidate profile
* Job role requirements

Output:

* Match score percentage

## Module 5: Skill Gap Analysis

Identify:

* Missing skills
* Missing tools
* Missing certifications

## Module 6: Career Roadmap

Recommend:

* Skills to learn
* Projects to build
* Certifications to pursue

## Module 7: Skill Demand Forecasting

Predict:

* Emerging skills
* Growing technologies
* Declining technologies

## Module 8: Dashboard

Provide:

* Market insights
* Career insights
* Skill intelligence

## Module 9: Streamlit Application

Allow users to:

* Upload resumes
* View recommendations
* Explore job market insights
