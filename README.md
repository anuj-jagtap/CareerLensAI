# CareerLens AI

AI-Powered Career Intelligence & Job Match Platform for Freshers

## Project Objective

Help freshers understand:
- Which IT roles suit their profile
- Which skills are in demand
- Why they may not be getting shortlisted
- What skills they should learn next

## Features

- Job Market Analytics
- Resume Intelligence
- Job Match Scoring
- Skill Gap Analysis
- Career Roadmap Generation
- Skill Demand Forecasting
- Power BI Dashboard
- Streamlit Application

## Tech Stack

- Python
- SQL
- Power BI
- Streamlit
- NLP
- Machine Learning
